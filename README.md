## Folder Structure
```

├── css/
│   └── style.css       ← All styles
├── js/
│   └── script.js       ← All JavaScript (animations, interactions)
└── images/             ← Add your photos here
    ├── dj/
    ├── shisha/
    ├── cocktails/
    ├── rooftop/
    └── food/
```

## Setup
1. Open `index.html` in any browser — no build tools needed.
2. Replace gallery placeholders with real images by editing the gallery section in `index.html`.
3. Update the Google Maps embed URL with your real coordinates.
4. Update phone numbers and social media links as needed.

## Adding Real Images
In `index.html`, find the `.gallery-item` divs and replace the placeholder divs with:
```html
<img src="images/dj/photo1.jpg" alt="DJ Night" />
```
